Median demand proxy per bucket, with bootstrap 95% confidence intervals (1000 resamples, n >= 100 per bucket required). A finding is rejected if any pair of bucket CIs overlaps.

Built on Burla. Code: github.com/Burla-Cloud/airbnb-burla. Caveats: demand proxy is reviews_per_month, not actual bookings. Photo coverage is whatever public listing pages returned without anti-bot blocking. Public Inside Airbnb data, no API access.
